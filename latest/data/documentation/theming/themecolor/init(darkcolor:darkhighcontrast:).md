# init(darkColor:darkHighContrast:)

Create a new theme color instance.

```
init(darkColor: Color, darkHighContrast: Color)
```

## Parameters

`darkColor`

The color used in dark mode.

`darkHighContrast`

The color used in dark mode with high contrast enabled.
