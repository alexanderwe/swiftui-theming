# init(name:colors:fonts:)

Create a new theme.

```
init(name: String, colors: Theme.ColorMap, fonts: Theme.FontMap = .appleDefault)
```

## Parameters

`name`

The name of the themes.

`colors`

The colors of the theme.

`fonts`

The fonts of the theme. If no fonts are specified the default Apple font styles will be used.
