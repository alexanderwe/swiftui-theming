# init(lightColor:lightHighContrast:darkColor:darkHighContrast:)

Create a new theme color instance.

```
init(lightColor: Color, lightHighContrast: Color? = nil, darkColor: Color, darkHighContrast: Color? = nil)
```

## Parameters

`lightColor`

The color used in light mode.

`lightHighContrast`

The color used in light mode with high contrast enabled.

`darkColor`

The color used in dark mode.

`darkHighContrast`

The color used in dark mode with high contrast enabled.
